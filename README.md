# 案件応募自動化システム

## まずこれを読んでください

**このリンクをClaude Codeに貼り付けてください。**

```
https://github.com/flight4x4/job-application-automation
```

「セットアップして」と一言送るだけで、あとはClaude Codeが全部案内します。

---

## できること

Coworkに一言送るだけで、案件の**検索・選別・応募・記録**が自動で動きます。

```
あなた：「クラウドワークスのGAS・業務自動化カテゴリで応募して」
　　↓
Cowork が候補案件を一覧表示
　　↓
あなた：「①③でいいよ」
　　↓
Cowork が応募文を自動生成して応募
　　↓
結果がGoogleスプレッドシートに自動記録
```

---

## セットアップ手順

### 1. Claude Codeに上のリンクを食べさせる

Claude CodeにURLを貼り付けて「セットアップして」と送るだけです。

### 2. 聞かれた情報を答えるだけ

Claude Codeが順番に必要なことを聞いてきます。
答えていくだけでセットアップが完了します。

> 技術的な作業（Google連携の設定・ファイル生成など）はClaude Codeがすべて行います。
> あなたが手動でやることはほぼありません。

---

## ⚠️ 1点だけ注意

**Coworkに指示を出す前に、クラウドワークスなどのサイトにブラウザでログインしておいてください。**

Coworkはあなたのブラウザを使って動きます。
ログアウトしていると動作しません。
